# Design Specification: Perspectives (Digital Garden)

**Purpose:**
A repository for technical inquiries, ethical ruminations, and intersections between AI and biological systems.

## Layout & Components

### Page Header

- **Headline:** "Perspectives" in bold Space Grotesk, half-white/half-pink styling.

### List View Architecture

- Articles presented as vertical rows with clear category filters (All Garden, AI Solutions, Bio-Medical, Ethics).
- **Metadata:** Read time and publication date positioned as lead metadata.

### Typography & Spacing

- High whitespace between article rows to ensure readability.
- Category tags use lowercase monospace text for a "digital garden" aesthetic.
- Interactive rows with neon purple glow on hover.
